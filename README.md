# 老版 QQ 聊天记录解密还原 Skill

## 项目介绍

在尝试恢复一份 2010 年的 QQ 聊天记录备份时，我发现目前已经很难找到仍能正常使用的读取工具。部分旧工具只能识别会话列表和联系人，却无法解密真正的聊天正文，相关文件格式资料也十分零散。

借助 Codex 对备份文件结构、`Index.msj`、`Data.msj`、QQ TEA 加密方式以及消息记录格式进行分析，最终成功解密并还原了这份老 QQ 聊天记录。

为了让这套方法能够被重复使用，我将完整流程整理成了 `recover-legacy-qq-chat` Skill，并分享出来。

## 主要功能

- 识别旧版 QQ MsgEx/OLE 备份结构
- 探测并验证可能的解密密钥
- 解密和解析私聊、群聊消息
- 清理旧 QQ 字体标记和富文本控制字符
- 将图片、表情引用整理为统一占位符
- 导出 CSV、TXT 和 QQChatExporter 风格 JSON
- 校验消息数量及解密完整性
- 对账号、昵称、密钥和本地路径进行脱敏

## 使用说明

项目中不包含任何真实聊天记录、QQ 账号、密钥或个人数据。

本项目仅用于恢复本人所有或已获得明确授权的聊天备份。请勿将其用于未经授权的数据访问。

## 一句话简介

借助 Codex 分析并还原 2010 年前后的旧版 QQ 加密聊天备份，并将完整流程整理成可复用、可分享的 Skill。
